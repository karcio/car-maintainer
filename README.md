# car-maintainer


## Clone repository
```
git clone git@github.com:karcio/car-maintainer.git
```

## Create an environment
```
cd car-maintainer
python3 -m venv venv
```

## Activate the environment
```
. venv/bin/activate
```

## Install requirements
```
pip3 install -r requirements.txt
```

## Run app
```
flask --app app run
```
